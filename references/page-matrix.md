# Page Matrix

## Purpose

Choose default pages by website type so the operator does not need to remember common website requirements.

## Rules

- Default to a multi-page static site, not only a landing page.
- Keep all pages plain static HTML.
- Link all selected pages in header or footer navigation.
- If the operator excludes a recommended page, record that it was intentionally excluded in the final checklist.

## Default Pages by Site Type

### Enterprise Website

Required by default:

- `index.html`: homepage
- `about.html`: company introduction
- `services.html`: services or solutions
- `contact.html`: contact page
- `privacy.html`: privacy policy draft or user-provided content
- `terms.html`: user agreement or terms draft/user-provided content

Optional:

- `cases.html`: case studies
- `pricing.html`: pricing or packages
- `news.html`: announcements
- `careers.html`: recruiting

### Product or Service Landing Site

Required by default:

- `index.html`
- `contact.html`
- `privacy.html`
- `terms.html`

Optional:

- `features.html`
- `pricing.html`
- `faq.html`
- `download.html`

### APP Website

Required by default:

- `index.html`
- `support.html`
- `privacy.html`
- `terms.html`
- `contact.html`

Optional:

- `download.html`
- `release-notes.html`
- `faq.html`

### Game or Game Developer Website

Required by default:

- `index.html`
- `press-kit.html`
- `support.html`
- `privacy.html`
- `terms.html`
- `contact.html`

Optional:

- `media.html`
- `community.html`
- `download.html`
- `updates.html`

### Data, Ads, Analytics, or Marketing Technology Website

Required by default:

- `index.html`
- `services.html`
- `contact.html`
- `privacy.html`
- `terms.html`

Recommended optional:

- `cookie-policy.html`
- `data-compliance.html`
- `security.html`

## Final Checklist Fields

Track these before packaging:

- Required pages created
- Optional pages selected
- Pages intentionally excluded
- Placeholder pages still present
- Legal content source: user-provided, generated draft, or pending
- Navigation and footer links updated
